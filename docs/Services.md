# Services

## Services List
---
```
user/{user_id}/services
```
| Args | Type | Example         |
|------|------|-----------------|
|      |List  |[spotify, google,...]|

## Service Status
---
```
user/{user_id}/services/{service_id}
```
| Args | Type | Example   |
|------|------|-----------|
|Status|String|Connect    |

## Service Connect
---
```
user/{user_id}/services/{service_id}/login
```
| Args | Type | Example   |
|------|------|-----------|
|      |String|Succes     |

## Add Service

| Args | Type | Example   |
